# Tech Stack

## Backend Technology Stack

### Primary Framework
- **.NET Framework 4.8** - Primary development platform
- **C#** - Programming language

### Database & Data Access
- **PostgreSQL** - Primary database
- **PostGIS** - Spatial database extension for geographic data
- **Entity Framework 6.x** - ORM compatible with .NET Framework 4.8
- **Npgsql** - PostgreSQL data provider for .NET

### Web Framework
- **ASP.NET Web API 2** - RESTful API framework for .NET Framework 4.8
- **OWIN/Katana** - Web server interface (if needed)

### Spatial Data Processing
- **NetTopologySuite** - .NET spatial library for geometric operations
- **GeoAPI** - Geographic API for .NET

## Frontend Technology Stack

### Framework & Libraries
- **React** - Frontend JavaScript library
- **MapLibre GL JS** - Map rendering library
- **TypeScript** - Type-safe JavaScript development

### Build & Tooling
- **Node.js** - JavaScript runtime for build tools
- **npm/yarn** - Package management
- **Webpack** - Module bundler

## Infrastructure & Deployment

### Cloud Platform
- **Microsoft Azure** - Primary cloud platform
- **Azure App Service** - Web application hosting
- **Azure Database for PostgreSQL** - Managed database service

### CI/CD & Version Control
- **GitHub** - Source code repository
- **GitHub Actions** - Continuous integration/deployment

### Monitoring & Observability
- **Azure Application Insights** - Application performance monitoring
- **Azure Monitor** - Infrastructure monitoring

## External Services

### Weather Data
- **Yr/Met.no API** - Primary weather service for Norwegian data
- **locationforecast.no** - Fallback weather service

### Geographic Data
- **Lantmäteriet** - Swedish geographic data source
- **OpenStreetMap** - Building footprint data source

## Development Tools

### IDE & Editors
- **Visual Studio 2019/2022** - Primary IDE for .NET Framework 4.8
- **Visual Studio Code** - Lightweight editor for frontend development

### Database Tools
- **pgAdmin** - PostgreSQL administration
- **PostGIS** - Spatial database extension tools

### Testing Frameworks
- **NUnit** - Unit testing framework for .NET Framework 4.8
- **Moq** - Mocking framework for unit tests
- **Jest** - JavaScript testing framework for frontend
